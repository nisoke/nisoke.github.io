收敛性理论

4.1 辅助引理

范数等价性引理

引理描述

设 $\Omega \subset \mathbb{R}^n$ 是具有 Lipschitz 边界的有界区域。令 $N = \dim \mathbb{P}{m-1}(\Omega) \ge 1$，其中 $\mathbb{P}{m-1}$ 是 $\Omega$ 上次数不超过 $m-1$ 的多项式函数空间。

设 $f_i: \mathbb{H}^m(\Omega) \rightarrow \mathbb{R}$，$i = 1, \ldots, N$ 是连续线性泛函，且满足：
$$\forall p \in \mathbb{P}_{m-1}(\Omega),\quad f_i(p) = 0 \ \text{当且仅当}\ p = 0.$$

则空间 $\mathbb{H}^m(\Omega)$ 上的范数 $|\cdot|{\mathbb{H}^m(\Omega)}$ 与
$$|\cdot|{\mathbb{H}^m(\Omega)} + \sum_{i=1}^N |f_i(\cdot)|$$
等价。即存在常数 $C_1, C_2 > 0$ 使得：
$$C_1|v|{\mathbb{H}^m(\Omega)} \le |v|{\mathbb{H}^m(\Omega)} + \sum_{i=1}^N |f_i(v)| \le C_2|v|_{\mathbb{H}^m(\Omega)},\quad \forall v \in \mathbb{H}^m(\Omega).$$

说明

${f_i}$ 在 $\mathbb{P}{m-1}’$ 上的限制构成了 $\mathbb{P}{m-1}$ 上的一组自由度，它们能唯一确定 $\mathbb{P}_{m-1}$ 中的元素（即分离不同的多项式）。

因此，这个引理说明了 $\mathbb{H}^m(\Omega)$ 中函数的范数可以由半范数加上一组具有特殊性质的泛函来控制。这是后续误差估计的基础。

Deny-Lions 引理

引理描述

设 $\Omega \subset \mathbb{R}^n$ 是具有 Lipschitz 边界的有界区域。对任意 $m \ge 1$，存在常数 $C = C(m, \Omega) > 0$ 使得：
$$\inf_{p \in \mathbb{P}{m-1}(\Omega)} |v - p|{\mathbb{H}^m(\Omega)} \le C|v|_{\mathbb{H}^m(\Omega)},\quad \forall v \in \mathbb{H}^m(\Omega).$$

说明

在具有 Lipschitz 边界的有界区域上，任意函数 $v \in \mathbb{H}^m(\Omega)$ 可以被 $\mathbb{P}{m-1}(\Omega)$ 中的多项式 $p$ 以 $C|v|{\mathbb{H}^m(\Omega)}$ 的精度逼近。

证明思路：由范数等价性引理，存在一组泛函 ${f_i}{i=1}^N$ 能唯一确定 $\mathbb{P}{m-1}$ 中的元素。选取 $p \in \mathbb{P}{m-1}$ 使得 $f_i(p) = f_i(v)$ 对 $i = 1, \ldots, N$ 成立（即 $p$ 是 $v$ 在 $\mathbb{P}{m-1}$ 上关于泛函 ${f_i}$ 的插值）。由于 $|\mathbb{P}{m-1}|{\mathbb{H}^m(\Omega)} = 0$（多项式的 $m$ 阶导数为零），可得：
$$|v - p|{\mathbb{H}^m(\Omega)} \le C\left(|v-p|{\mathbb{H}^m(\Omega)} + \sum_{i=1}^N |f_i(v-p)|\right) = C|v|_{\mathbb{H}^m(\Omega)}.$$

4.2 尺度变换

仿射等价

定义

称两个单元 $K$ 和 $\hat{K}$ 是仿射等价的，若存在可逆仿射变换 $F: \mathbb{R}^d \rightarrow \mathbb{R}^d$ 使得 $F(\hat{K}) = K$，即：
$$x = B\hat{x} + b,\quad \forall \hat{x} \in \hat{K},$$
其中 $B \in \mathbb{R}^{d \times d}$ 是可逆矩阵，$b \in \mathbb{R}^d$ 是平移向量。

相应地，定义在 $K$ 上的函数 $v$ 与定义在 $\hat{K}$ 上的函数 $\hat{v}$ 满足：
$$\hat{v}(\hat{x}) = v(F(\hat{x})) = v(x),\quad \text{或等价地}\quad v(x) = \hat{v}(F^{-1}(x)).$$

半范数在仿射变换下的估计

定理

设 $K$ 与 $\hat{K}$ 仿射等价，变换为 $x = B\hat{x} + b$。对任意 $v \in \mathbb{H}^m(K)$，令 $\hat{v} = v \circ F \in \mathbb{H}^m(\hat{K})$，则存在仅依赖于 $d$ 和 $m$ 的常数 $C > 0$，使得：
$$|v|{\mathbb{H}^m(K)} \le C |B^{-1}|^m |\det B|^{1/2} |\hat{v}|{\mathbb{H}^m(\hat{K})},$$
$$|\hat{v}|{\mathbb{H}^m(\hat{K})} \le C |B|^m |\det B|^{-1/2} |v|{\mathbb{H}^m(K)}.$$

其中 $|\cdot|$ 表示矩阵的欧几里得范数，常数 $C$ 通常可取为 $C = d^m$（依赖于维数 $d$ 和阶数 $m$）。

证明概述

由链式法则，$m$ 阶导数满足：
$$\partial_{i_1} \cdots \partial_{i_m} \hat{v}(\hat{x}) = \sum_{j_1, \ldots, j_m = 1}^d B_{j_1 i_1} \cdots B_{j_m i_m} \partial_{j_1} \cdots \partial_{j_m} v(x),$$
其中 $B_{ji}$ 是矩阵 $B$ 的元素。对上式两边取平方并在 $\hat{K}$ 上积分，利用 Cauchy-Schwarz 不等式可得：
$$|\hat{v}|{\mathbb{H}^m(\hat{K})}^2 \le C \sum{j_1, \ldots, j_m = 1}^d |B e_{j_1}|^2 \cdots |B e_{j_m}|^2 \cdot |v|_{\mathbb{H}^m(K)}^2 |\det B|^{-1}.$$
进一步利用 $|B e_j| \le |B|$ 放缩，得到所需估计。

几何估计

定理

设 $K$ 与 $\hat{K}$ 仿射等价，变换为 $x = B\hat{x} + b$。记：

  • $h_K = \operatorname{diam}(K)$，$h_{\hat{K}} = \operatorname{diam}(\hat{K})$ 分别为 $K$ 和 $\hat{K}$ 的直径；
  • $\rho_K$，$\rho_{\hat{K}}$ 分别为 $K$ 和 $\hat{K}$ 的内接球半径。

则成立以下估计：
$$|B| \le \frac{h_K}{\rho_{\hat{K}}},\quad |B^{-1}| \le \frac{h_{\hat{K}}}{\rho_K},\quad |\det B| = \frac{|K|}{|\hat{K}|},$$
其中 $|K|$ 和 $|\hat{K}|$ 分别表示 $K$ 和 $\hat{K}$ 的体积（或测度）。

4.3 插值理论

局部插值估计

局部插值算子

设 $\mathcal{I}h: \mathbb{H}^m(K) \rightarrow \mathbb{P}{k}(K)$ 是定义在单元 $K$ 上的插值算子，满足：

  • $\mathcal{I}h p = p$，$\forall p \in \mathbb{P}{k}(K)$（多项式保持性）；
  • $\mathcal{I}_h$ 是连续线性算子。

局部误差估计

在仿射等价变换下，可以得到局部插值误差估计：
$$|v - \mathcal{I}h v|{\mathbb{H}^m(K)} \le C h_K^{k+1-m} |v|_{\mathbb{H}^{k+1}(K)},\quad \forall v \in \mathbb{H}^{k+1}(K),$$
其中 $C$ 是与 $K$ 无关的常数，仅依赖于参考单元 $\hat{K}$ 和插值算子的性质。
